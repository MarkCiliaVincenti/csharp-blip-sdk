*** This is an experimental template! Use at your own risk (but please give your feedback on https://github.com/takenet/messaginghub-client-csharp/issues)

Navigate to https://blip.ai to retrieve your application's Webhook key, and set it on appSettings in Web.config.
Just add a reference to you library project (using C# SDK host template) so it can run as a webhook template.

See more information about how to build your application at https://portal.blip.ai/#/docs.
